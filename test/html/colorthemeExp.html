colortheme颜色主题

切换layui的颜色主题,提案PR #1410

写在前面

注意事项

  • * colortheme仅对2.8版本的layui.css中的颜色进行归类,并使用css变量实现颜色主题的切换;仅对相同的颜色进行合并定义,在颜色的归纳上并不算系统

  • * colortheme是将layui.css中的颜色重新使用css变量进行定义

    1. 1、 对于颜色的样式会有重复
    2. 2、 没有把握颜色的权重,因此会引起颜色样式的问题;对于这个问题是发现一个改正一个
    3. 3、 这个功能是基于layuiAdmin的主题设置,提供了一些方法,需要自己去实现自定义的功能
    4. 4、 由于上面的各种问题,colortheme功能需要手动开启,在引入css和js文件之后,调用layui.colortheme.run()开启功能

方法

在开启colortheme功能后,会去缓存里面获取主题信息,没有就设置成默认的主题

colortheme功能使用layui.data操作缓存,缓存的key可以通过:layui.config({themeKey:'XXX'});layui.config({themeConfigKey:'XXX'});来进行修改,用来区分不同网站的key,防止混淆

方法介绍
方法名 描述 参数 返回值
getTheme 获取主题色的key(就是themeKey对应的缓存) - 当前的themeKey对应的缓存
setTheme 设置主题 - 参数是指定一个themeKey(没有参数就去缓存中的) string -
getConfig 获取主题配色方案(就是themeConfigKey对应的缓存) - 主题配色方案对象
setConfig 自定义配色方案(name:方案中文名称;cnname:方案英文名称;config:方案的颜色配置) name, cnname, config -
popup 弹出一个右侧的抽屉层,这个是内置的可视化配置界面,是模拟的layuiAdmin的主题设置 - -

自定义主题

当前默认有:default(默认配色);classicBlackHeader(经典黑头);darkBlue(藏蓝);coffee(咖啡);purpleRed(紫红);ocean(海洋);green(墨绿);fashionRed(时尚红);这些样式,设置时传入的key就是英文名,括号里面的是中文名

可以自定义主题,需要传入英文名、中文名、和配置项,传入的英文名和原有的重复就是修改原来的主题设置。配置项就是一个key-value的对象,值是颜色的rgb值,可以使用下面的方式,使用layui的颜色选择组件来选择

layui.colorpicker.render({
    elem: "#theme",
    format: "rgb",
    done: function (color) {
      // 选择颜色后将值反填到输入框中,因为最后上传的是表单中输入框里面的信息
      $("#form").val(color.substring(4, color.length - 1));
    },
});

其它

类似于layuiframework,colortheme的使用一般是要控制整个系统的,但是当前能影响的只是当前页面。所以这个功能也需要自己去实现

下面是layuiframework中实现多个iframe之前联动主题的方式(iframe需要先引用并启用colortheme功能)

// 主题监听事件修改,就是修改当前的colortheme修改主题的方法,在修改完之后也要去查找各个iframe页面去调用他们的方法来修改主题
// touchLayuiMoudle 方法是确定这个页面里面的layui.colortheme加载成功之后再调用方法
const setTheme = layui.colortheme.setTheme;
const setConfig = layui.colortheme.setConfig;
layui.colortheme.setTheme = function(key){
    setTheme(key);
    // 修改首页
    var iframe = $bodyGroup.find('[lay-id="0"]').find("." + ELEM_IFRAME_CLASS).get(0);
    handler.touchLayuiMoudle(function () {
        return (
            iframe &&
            iframe.contentWindow &&
            iframe.contentWindow.layui &&
            iframe.contentWindow.layui.colortheme
        );
    }, function(){
        iframe.contentWindow.layui.colortheme.setTheme(key)
    });

    // 修改普通页面
    _lodash.each(handler.cache.cache, (v, k) => {
        let _iframe = $bodyGroup.find('[lay-id="'+ v.value +'"]').find("." + ELEM_IFRAME_CLASS).get(0);
        handler.touchLayuiMoudle(function () {
            return (
                _iframe &&
                _iframe.contentWindow &&
                _iframe.contentWindow.layui &&
                _iframe.contentWindow.layui.colortheme
            );
        }, function(){
            _iframe.contentWindow.layui.colortheme.setTheme(key)
        });
    });
};

layui.colortheme.setConfig = function(name, cnname, config){
    setConfig(name, cnname, config);
    // 修改首页
    var iframe = $bodyGroup.find('[lay-id="0"]').find("." + ELEM_IFRAME_CLASS).get(0);
    handler.touchLayuiMoudle(function () {
        return (
            iframe &&
            iframe.contentWindow &&
            iframe.contentWindow.layui &&
            iframe.contentWindow.layui.colortheme
        );
    }, function(){
        iframe.contentWindow.layui.colortheme.setConfig(name, cnname, config)
    });

    // 修改普通页面
    _lodash.each(handler.cache.cache, (v, k) => {
        let _iframe = $bodyGroup.find('[lay-id="'+ v.value +'"]').find("." + ELEM_IFRAME_CLASS).get(0);
        handler.touchLayuiMoudle(function () {
            return (
                _iframe &&
                _iframe.contentWindow &&
                _iframe.contentWindow.layui &&
                _iframe.contentWindow.layui.colortheme
            );
        }, function(){
            _iframe.contentWindow.layui.colortheme.setConfig(name, cnname, config)
        });
    });
};
