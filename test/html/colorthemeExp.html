颜色主题

颜色主题colortheme是对2.8版本的layui.css中的颜色进行归类。使用css变量实现颜色主题的切换;仅对相同的颜色进行合并定义。PR #1410

  • 1、 合并颜色的样式会有重复,没有计较颜色的权重,对于样式冲突问题会持续完善。
  • 2、 这个可视化配置功能是参照layuiAdmin的主题设置
  • 3、 colortheme功能需要手动开启,在引入css和js文件之后,调用layui.colortheme.run()开启功能

内置主题

当前默认有:default(默认配色);classicBlackHeader(经典黑头);darkBlue(藏蓝);coffee(咖啡);purpleRed(紫红);ocean(海洋);green(墨绿);fashionRed(时尚红);这些样式,设置时传入的key就是英文名,括号里面的是中文名

可以自定义主题,需要传入英文名、中文名、和配置项,传入的英文名和原有的重复就是修改原来的主题设置。配置项就是一个key-value的对象,值是颜色的rgb值,可以使用下面的方式,使用layui的颜色选择组件来选择

方法

颜色主题初始化

layui.colortheme.run();可以对当前页面的主题进行初始化操作

颜色主题缓存设置

colortheme使用layui.data做持久化处理。想要切换缓存的空间,可以调用layui.config({themeKey:'XXX'});

colortheme使用layui.data做持久化处理。想要切换缓存的空间,可以调用layui.config({themeConfigKey:'XXX'});

  • 参数XXX是需要修改的缓存的空间的名称,必填为了防止多个使用colortheme页面之间的冲突,建议给你的页面起一个专属的缓存空间的名称。
  • themeKey缓存的是主题的英文名,是一个字符串
  • themeConfigKey缓存的所有的配色方案,是一个数组

获取颜色主题

layui.colortheme.getTheme(key);获取这个key对应的颜色主题的英文名

  • 参数key可选,是颜色主题的英文名称(在内置主题一栏有介绍内置的英文名称和中文名称),如果不传就返回当前正在使用的颜色主题的英文名

设置颜色主题

layui.colortheme.setTheme(key);将当前页面的主题设置成这个key对应的颜色主题(该主题配色方案需要在配置缓存中存在)

  • 参数key可选,是颜色主题的英文名称(在内置主题一栏有介绍内置的英文名称和中文名称),如果不传就使用当前正在使用的颜色主题的英文名

获取主题配色方案

layui.colortheme.getConfig();获取当前缓存的所有主题配色方案

新增/修改主题配色方案

layui.colortheme.setConfig(name, cnname, config);可以保存用户自定义的颜色配置方案。如果传入主题配色方案是一个新的英文名就在配置缓存中新增一个,如果不是就修改原来的主题配色方案

  • 参数name必填,是主题配色方案的英文名称(也是该主题配色方案的唯一id)
  • 参数cnname必填,是主题配色方案的中文名称
  • 参数config必填,是主题配色方案的内容,内容是对上面layui颜色主题css变量定义的自定义声明

在自定义颜色时可以使用layui的颜色选择组件来做,下面是一个例子

layui.colorpicker.render({
    elem: "#theme",
    format: "rgb",
    done: function (color) {
      // 选择颜色后将值反填到输入框中,因为最后上传的是表单中输入框里面的信息
      $("#form").val(color.substring(4, color.length - 1));
    },
});

其它

使用上面的方式只能修改当前页面的主题,如果要想多个页面随着修改,可以试试下面的方式。页面上方的主题设置就是用的这种方式

// 主题监听事件修改,就是修改当前的colortheme修改主题的方法,在修改完之后也要去查找各个iframe页面去调用他们的方法来修改主题
// touchLayuiMoudle 方法是确定这个页面里面的layui.colortheme加载成功之后再调用方法
const setTheme = layui.colortheme.setTheme;
const setConfig = layui.colortheme.setConfig;
layui.colortheme.setTheme = function(key){
    setTheme(key);
    // 修改首页
    var iframe = $bodyGroup.find('[lay-id="0"]').find("." + ELEM_IFRAME_CLASS).get(0);
    handler.touchLayuiMoudle(function () {
        return (
            iframe &&
            iframe.contentWindow &&
            iframe.contentWindow.layui &&
            iframe.contentWindow.layui.colortheme
        );
    }, function(){
        iframe.contentWindow.layui.colortheme.setTheme(key)
    });

    // 修改普通页面
    layui.each(handler.cache.cache, (k, v) => {
        let _iframe = $bodyGroup.find('[lay-id="'+ v.value +'"]').find("." + ELEM_IFRAME_CLASS).get(0);
        handler.touchLayuiMoudle(function () {
            return (
                _iframe &&
                _iframe.contentWindow &&
                _iframe.contentWindow.layui &&
                _iframe.contentWindow.layui.colortheme
            );
        }, function(){
            _iframe.contentWindow.layui.colortheme.setTheme(key)
        });
    });
};

layui.colortheme.setConfig = function(name, cnname, config){
    setConfig(name, cnname, config);
    // 修改首页
    var iframe = $bodyGroup.find('[lay-id="0"]').find("." + ELEM_IFRAME_CLASS).get(0);

        let _iframe = $bodyGroup.find('[lay-id="'+ v.value +'"]').find("." + ELEM_IFRAME_CLASS).get(0);
        handler.touchLayuiMoudle(function () {
            return (
                _iframe &&

            _iframe.contentWindow.layui.colortheme.setConfig(name, cnname, config)
